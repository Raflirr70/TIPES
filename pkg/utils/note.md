Di folder utils, digunakan untuk menyimpan fungsi-fungsi pembantu (helper functions) yang dapat digunakan di berbagai bagian aplikasi.
Contoh :

- Fungsi untuk format tanggal
- Fungsi untuk mengirim email
- Fungsi untuk generate random string
- Fungsi untuk Generate Kursi pesawat

  Yang perlu di folder utils :

- Fungsi-fungsi yang bersifat umum dan dapat digunakan di berbagai bagian aplikasi
- Fungsi-fungsi yang tidak tergantung pada konteks bisnis aplikasi

  Yang tidak perlu di folder utils :

- Logic bisnis aplikasi (letakkan di service)
